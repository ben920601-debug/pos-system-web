# POS 系統 Web 版 (POS System Web)

這是一個現代化、前後端分離的 Web POS 銷售系統。
採用原生 JavaScript 模組化開發前端，並使用 C# ASP.NET Core 打造穩健的後端 API，實現了從收銀、庫存管理到交班結算的完整銷售流程。

## 🚀 專案簡介

本系統旨在提供一個輕量、快速且具備完整帳務邏輯的收銀解決方案。介面設計仿照原生 App 操作體驗，具備全螢幕鎖定與 RWD 響應式佈局，適合在觸控螢幕或一般電腦上操作。

## ✨ 主要功能

### 1. 🔐 安全登入系統
* 帳號密碼驗證 (連接資料庫)。
* 權限檢查與 Session 管理 (防止未登入操作)。
* 記錄登入時間以進行交班計算。

### 2. 🛒 收銀台 (POS)
* **商品展示**：分類顯示商品，支援點擊加入。
* **購物車管理**：支援增減數量、刪除商品。
* **結帳計算**：即時計算總金額、輸入實收金額自動算找零。
* **介面優化**：左側選單固定，防止視窗捲動，提供 App 般的操作手感。

### 3. 📦 商品管理 (CRUD)
* 查看所有商品列表。
* **新增、修改、刪除**商品資料 (即時同步資料庫)。
* 支援修改價格、名稱與分類。

### 4. 📄 歷史訂單
* 查詢過往所有交易紀錄。
* 支援主從式介面 (Master-Detail)，點擊單號即可查看詳細商品明細。

### 5. 🔄 交班結算 (Shift Management)
* **實點對帳 (Blind Balancing)**：店員需輸入錢櫃實有金額，系統自動比對銷售額。
* **短溢計算**：自動計算帳務是否吻合 (溢收/短少)。
* **預備金管理**：支援設定預設找零金，並正確顯示於報表中。
* **交班紀錄**：結算後自動存檔至資料庫並登出。

## 🛠️ 技術架構

### 前端 (Frontend)
* **核心語言**：HTML5, CSS3, JavaScript (ES6+)。
* **架構設計**：ES Modules (模組化開發)，無依賴大型框架 (Vanilla JS)。
* **樣式佈局**：CSS Grid & Flexbox，實現全螢幕固定佈局。
* **API 串接**：使用 Fetch API 與後端進行非同步通訊。

### 後端 (Backend)
* **框架**：C# ASP.NET Core Web API。
* **資料庫**：PostgreSQL。
* **ORM**：Entity Framework Core。

## 🚀 如何執行

1.  **啟動後端**：
    ```bash
    cd api
    dotnet run
    ```
    *(後端預設 Port: 5099)*

2.  **啟動前端**：
    ```bash
    # 回到根目錄
    node server.js
    ```
    *(瀏覽器將自動開啟 http://localhost:3000)*
